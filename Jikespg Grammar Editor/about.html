About the Grammar Editor

July 5th, 2005

Abstract

The Grammar Editor is a full featured Eclipse Eclipse editor. It can be used to edit grammar files that act as input for the Jikes Parser Generator or the LALR Parser Generator.

Features

It supports:

  • Content assistance when editing grammar elements, macros or options
  • Editor outline
  • Quick outline
  • Rename in file
  • Find occurrences in file on element basis
  • Element, macro and included file source navigation and lookup using hovers
  • Element folding
  • Syntax and semantic highlighting
  • Syntax and semantic check while typing
